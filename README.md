## Quick Overview / 快速概览

Claude Code uses a dual rate limit system:
- **5-hour limits**: Reset every 5 hours
- **Weekly limits**: Reset every 7 days (effective August 28, 2025)

Claude Code 采用双重速率限制系统：
- **5小时限制**：每5小时重置
- **每周限制**：每7天重置（2025年8月28日生效）
